FoBo <...data-lift="Front-End Toolkit Module"/>

This pages demonstrates a optional Lift, generic front-end toolkit module FoBo that quick and easy will integrate some of the industry leading web-centric open source front-end toolkits (like twitter bootstrap and more) into your Lift application, giving you the flexibility and choice of setting up and use the toolkit:s of your liking.

FoBo provides smoth toolkit upgrades by minimizing the point of change with uniform template declarations. Added to this is a growing number of FoBoAPI functions that will gives you even more ease of usage, power and control.

Except from reading the BootstrapSetupUsage specific page or the more general Setup section below and the FoBoAPI the most up to date way to explore and learn how to use the FoBo module would be to fork and/or study the Templating With Twitter Bootstrap it is a basic Lift starter template project that includes FoBo for it's Bootstrap support. You can also study this demo code on Github.

For more information about the FoBo module take a look at the FoBoAPI and the readme on Github.

Also check out the integrated toolkit documentation for a comprehensive list of the, in this demo included, toolkits features.

Mobile Scalability with FoBo

FoBo comes with setup options for several front end toolkits including ZURB/Foundation, Twitter Bootstrap and JQuery Mobile. ZURB/Foundation and Twitter Bootstrap are both great reactive and media scalable toolkits. The JQuery Mobile toolkit is a awsome Touch-Optimized Web Framework for Smartphones & Tablets.

Knockout JS

Knockout - Simplify dynamic JavaScript UIs by applying the Model-View-View Model (MVVM) pattern. Some of Knockouts key concepts is:

  • Declarative Bindings - Easily associate DOM elements with model data using a concise, readable syntax.
  • Automatic UI Refresh - When your data model's state changes, your UI updates automatically.
  • Dependency Tracking - Implicitly set up chains of relationships between model data, to transform and combine it.
  • Templating - Quickly generate sophisticated, nested UIs as a function of your model data.
