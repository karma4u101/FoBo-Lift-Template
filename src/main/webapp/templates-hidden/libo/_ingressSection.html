The FoBo module and Bootstrap

This is work in progress!

The FoBo module gives your Lift project seamless front-end toolkit integration. This page focus on the FoBo/Lift Twitter-Bootstrap integration, including general bootstrap usage and information on how to use some of FoBo's API functions.

Although the benefits of using the FoBo module is several without even using the FoBo Bootstrap API hooks the sections below will take you through the FoBo setup to enable Bootstrap in your Lift project and how you can make use of the API functions.
